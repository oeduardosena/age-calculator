== Description ==
O **Age Calculator** é um plugin simples para WordPress que calcula a idade com base na data de nascimento fornecida via shortcode.

== Installation ==
1. Faça o upload da pasta do plugin para `/wp-content/plugins/` ou instale diretamente pelo painel do WordPress.
2. Ative o plugin na aba "Plugins" do WordPress.

== Usage ==
Use o seguinte shortcode em posts ou páginas:
`[time_ago birthdate="1990-01-01"]`
Formato AAAA-MM-DD
Isso exibirá a idade baseada na data fornecida.
